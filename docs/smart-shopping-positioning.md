# AffiliPilot smart-shopping positioning

## Product identity

AffiliPilot is a semi-automated affiliate assistant for **mua sắm thông minh**.

It helps the operator find, filter, draft, approve, and publish shopping posts with control. The system should not be anchored to one old niche such as mother/baby, nor to one specific Facebook Page name.

## Positioning

> Mua sắm thông minh — món nhỏ, tiện, đáng tiền, dễ kiểm chứng.

## Default audience

People who want practical buying suggestions for everyday life:

- household utility
- daily consumables
- home organization
- kitchen tools
- cleaning tools
- small electronics/accessories
- personal care with safe claims
- mother/baby as an optional category, not the core identity

## Content style

- short, natural Vietnamese
- one concrete use case
- one reason the product is worth checking
- no medical/unsafe claims
- no fake review tone
- no generic BA/checklist language
- affiliate disclosure stays short: `Link affiliate nhé 👇`

## Default caption shape

```text
<short title>

Món này hợp với <nhu cầu cụ thể>: <lợi ích đời thường>. Giá tham khảo trên <provider> khoảng <price>đ.

Link affiliate nhé 👇
#muasamthongminh #dogiadung #dealhot
```

## Tracking defaults

- property/profile: `smartshopping`
- UTM campaign prefix: `affilipilot_smart_shopping_YYYYMM`

Legacy page/category values may exist in old test data and archives, but new production runs should use smart-shopping profile defaults unless explicitly configured otherwise.
